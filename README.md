# Community Waste Management Tracker

A civic tech solution that empowers citizens to report waste issues and track garbage collection schedules in Ethiopia.

## Features

### For Citizens
- 📍 Report waste issues with photos and auto-detected location
- 🗺️ View all reports on an interactive map
- 📅 Check garbage collection schedules by sub-city
- 🔔 Receive notifications when report status changes
- 🗣️ Full Amharic language support

### For Administrators
- 📊 Dashboard to manage all reports
- ✅ Update report statuses (new, in-progress, resolved)
- 📝 Manage collection schedules
- 📈 View statistics and analytics

## Tech Stack

- **Frontend**: React.js
- **Maps**: Leaflet.js with OpenStreetMap
- **Backend**: Firebase (Firestore, Storage, Auth)
- **Styling**: CSS with modern variables
- **Language**: English & Amharic
